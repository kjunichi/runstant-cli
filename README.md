# 動かし方

## 準備

```bash
git clone https://github.com/kjunichi/runstant-cli.git
cd runstant-cli
npm install
```

## サンプルを動かす

### LinuxやOSXなど

```bash
cd examples
../bin/runstant-cli.js
```

### Windowsの場合

```bash
cd examples
node ../bin/runstant-cli.js
```


### もし気に入ったら、以下でrunstant-cliコマンドがインストールできます。

```
npm install -g
```
