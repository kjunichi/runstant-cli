# 動かし方

## 準備

```bash
git clone https://github.com/kjunichi/runstant-cli.git
cd runstant-cli
npm install
```

## サンプルを動かす

### LinuxやOSXなど

```bash
bin/runstant-cli.js examples
```

### Windowsの場合

```dos
node ..\bin\runstant-cli.js examples
```


### もし気に入ったら、以下でrunstant-cliコマンドがインストールできます。

```
npm install -g
```

Windowsでもrunstant-cliで実行可能となります。
